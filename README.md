# StudySprint 🚀

StudySprint is a full-stack student productivity application designed to help students manage tasks, goals, and focused study sessions.

## 🌐 Live Demo

**Live Website:** https://study-sprint-rosy.vercel.app

## 📂 GitHub Repository

https://github.com/namrathacn/StudySprint

---

## Features

### 🔐 Authentication
- Firebase Email/Password Authentication
- Protected routes
- Secure user sessions

### ✅ Task Management
- Create tasks
- View tasks
- Mark tasks completed
- Delete tasks
- User-specific tasks

### 🎯 Goal Tracking
- Create goals
- Track completion
- Delete goals
- User-specific goals

### ⏱️ Focus Timer
- Pomodoro-style study timer
- Saves completed study sessions
- Notification reminder support

### 📊 Dashboard
- Total tasks
- Completed tasks
- Goals progress
- Study sessions
- Progress tracking

### 👤 Profile
- User profile
- Study statistics

### ⚙️ Settings
- Theme preference
- Notification settings
- Logout

---
